Sistema de Gerenciamento do Projeto Clicando na Terceira Idade

Resumo

O presente trabalho aborda o desenvolvimento de um sistema que visa atender idosos, bolsistas e coordenadores do projeto Clicando na Terceira Idade desenvolvido pelo Instituto Federal Catarinense - Campus Avançado Sombrio. O projeto oferece curso de informática básica para os idosos do município de Sombrio - SC. Através deste projeto eles têm a oportunidade de conhecer o mundo da informática. O desenvolvimento de uma ferramenta digital para gerenciar as aulas se fez importante para que as edições atuais e anteriores do projeto possam ficar disponíveis para os participantes e assim serem acessadas a qualquer momento. Através do desenvolvimento deste sistema serão gerenciadas as aulas e todas as demais atividades desenvolvidas no âmbito do projeto.

Objetivos Geral e Específico

O objetivo deste projeto é criar um sistema de gerenciamento que beneficie todos os participantes, simplificando o acesso a materiais de aula, eventos, registros de atividades, horas dedicadas e promoção do projeto. Este sistema será adaptado para garantir facilidade de uso também para idosos.

Os nossos objetivos específicos são permitir o cadastro dos usuários, a visualização de usuários cadastrados, o login dos usuários, o cadastro de no máximo 20 alunos por turma, permitir o cadastro das turmas, permitir o lançamento e visualização de atividades, eventos e registros, o sistema deve apresentar dados do projeto como local, hora e data das aulas, apresentar a categoria dos usuários do sistema, apresentar a data de início e fim do projeto, permitir alterar dados do cadastro.

Tela de Login, onde os usuários irão colocar sua senha e seu usuário.

Tela de Menu, será a primeira tela que os usuários irão ter contato.

Tela de Cadastro, onde os usuários irão colocar seus dados para se cadastrarem.
